運動後可以吃什麼？黃伊婷 Doreen

那麼辛苦燃燒掉卡路里，肯定要堅持住不能吃東西，消耗體力又飢餓才會瘦！

ＮＧ！當然不是的！

運動後身體以及肌肉都處於極度失去能量的狀態，若再不補給營養的東西給身體充電，可是會讓身體無法修復還會越來越累唷！運動後絕對不要挨餓～只要選對東西吃，把握好時機補充，補給體力又不怕胖。但到底該吃些什麼呢？

運動後完美的補充以『碳水化合物＋蛋白質』有利於增加肌肉質量與性能，讓體力恢復，也可減少體脂肪產生。

碳水化合物

運動後補給優質碳水化合物，有助於體內血糖上升，會刺激胰島素的分泌，胰島素的水平提高能讓身體把養分經由血液循環送達到需要修復的肌肉組織，加速提升體力恢復。

蛋白質

蛋白質是肌肉增長的重要原料，可以修補與重建運動後遭破壞的肌肉組織。

運動後 聰明吃

  • 選擇1糙米飯 / 大約半碗份量(100g) + 雞胸肉 / 大約一個手掌份量(60g)(若要改成豬肉或牛肉，可選擇瘦的後(前)腿肉部位)
  • 選擇2馬鈴薯(90g) + 南瓜(85g) 或 玉米(2/3根) + 鮭魚 / 大約一個手掌份量(60g) 。
  • 選擇3熟麵條 / 1碗(120g) + 小方豆干/ 2塊半(100g) 。

( 上述選擇的碳水化合物均可對調互配，蛋白質也是，其比值為大約2:1 )

運動後 方便吃 （超商）

  • 選擇1御飯糰 + 茶葉蛋
  • 選擇2御飯糰 + 低糖豆漿
  • 選擇3牛奶 + 香蕉 + 茶葉蛋
  • 選擇4地瓜(大約110g) + 無糖豆漿
  • 選擇5全麥麵包 + 茶葉蛋 + 香蕉

（上述選擇均可對調互配， 食品後面營養標示很重要喔！）

健康運動，也要健康吃！別讓肚子不開心～
